# Vortex前端代码模板化生成
 
## 立意
减少重复繁琐的工作

## 面向群体
公司内部前端研发

## 结构
```
├── /cli/                  # 废弃
├── /dto/                  # 配置项服务 前端服务
├── /generate/             # 代码生成服务 
├── /generate2/            # 代码生成服务v2.0 （后续只维护这套）
├── .editorconfig          
├── README.md                
├── changelog.md           
```

## 使用方式
``` 
1.启动配置项服务
  dto目录下使用
  yarn install
  yarn start
2.启用代码生成服务
  generate目录下
  yarn install
  yarn start
```
